How it works & benchmarks

Standard MCP re-sends every tool schema on every turn. Code Mode exposes a single tool that accepts a TypeScript script — your agent writes all operations in one script, executes them in one round trip, and eliminates the repeated schema overhead. 

POST https://api.getbee.io/v2/sdk/mcp/codemode
Authorization: Bearer <access_token>
x-bee-template-id: <template-id>

Same auth, same template creation, same headers. Just swap the endpoint URL.
